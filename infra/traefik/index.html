Hotel Booking API

Point d’entrée unique (Traefik). Endpoints disponibles :

  • /auth — Authentification (login, register, refresh)
  • /auth/api — Swagger Auth
  • /users — Utilisateurs
  • /users/api — Swagger Users
  • /hotels — Hôtels et chambres
  • /hotels/api — Swagger Hotels
  • /reservations — Réservations
  • /reservations/api — Swagger Reservations
  • /payments — Paiements
  • /payments/api — Swagger Payments
  • /notifications — Notifications
  • /notifications/api — Swagger Notifications

Pour tester l’API : ouvrir le Swagger du service (ex. /auth/api). Les requêtes « Try it out » partent vers localhost:3000 (Traefik), donc tout fonctionne avec un seul point d’entrée.
